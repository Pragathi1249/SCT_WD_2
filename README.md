# Calculator Web Application

A fully functional Calculator Web Application built using **HTML, CSS, and JavaScript** in a single file. This project demonstrates **DOM manipulation**, **event handling**, **input parsing**, **keyboard interactions**, and **error handling** while providing a clean and responsive user interface.

## Features

### Basic Arithmetic Operations

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Percentage (%)

### User Interface

* Modern and minimal design
* Responsive layout for desktop, tablet, and mobile devices
* Smooth animations and hover effects
* Large display screen for expressions and results

### Functionalities

* Perform calculations using on-screen buttons
* Full keyboard support
* Clear (C) button to reset input
* Delete (⌫) button to remove the last character
* Decimal number support
* Real-time display updates

### Keyboard Shortcuts

| Key       | Action                |
| --------- | --------------------- |
| 0–9       | Enter numbers         |
| +         | Addition              |
| -         | Subtraction           |
| *         | Multiplication        |
| /         | Division              |
| %         | Percentage            |
| .         | Decimal point         |
| Enter     | Calculate result      |
| Backspace | Delete last character |
| Escape    | Clear calculator      |

### Error Handling

* Prevents invalid expressions
* Handles division by zero
* Displays error messages for malformed inputs
* Prevents consecutive operators
* Restricts multiple decimal points within a number

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/calculator-web-app.git
```

### Open the Project

1. Navigate to the project folder.
2. Open `index.html` in any modern web browser.

No installation, dependencies, or backend setup is required.
